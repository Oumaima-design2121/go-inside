1. Basic Definitions

Function Series Definition

A function series is an infinite sum of functions:

\[S(x) = \sum_{n=1}^{\infty} f_n(x) = f_1(x) + f_2(x) + f_3(x) + \cdots\]

The partial sum is: S_n(x) = f_1(x) + f_2(x) + ... + f_n(x)

Key Questions to Ask:

For which x does the series converge? (Domain of convergence)
What type of convergence do we have? (Pointwise, uniform, absolute, normal)
What properties does the sum function S(x) inherit? (Continuity, differentiability, integrability)
Can we interchange operations? (lim ‚àë = ‚àë lim, d/dx ‚àë = ‚àë d/dx, ‚à´‚àë = ‚àë‚à´)

2. Types of Convergence

Pointwise Convergence

Definition: For each fixed x, S_n(x) ‚Üí S(x) as n ‚Üí ‚àû

Symbolically: ‚àÄx, ‚àÄŒµ>0, ‚àÉN(Œµ,x) such that n>N ‚áí |S_n(x)-S(x)|<Œµ

Symbolically: ‚àÄŒµ>0, ‚àÉN(Œµ) such that ‚àÄx, n>N ‚áí |S_n(x)-S(x)|<Œµ

Definition: ‚àë|f_n(x)| converges for each x

Normal Convergence

Key feature: Strongest type of convergence

‚úÖ Implies: Uniform + Absolute convergence

3. Complete Set of Convergence Tests

For Pointwise Convergence:

Test 1: Fix x and Use Numerical Series Tests

Treat x as constant and apply:

  • Ratio Test: lim|f_{n+1}(x)/f_n(x)| < 1
  • Root Test: lim|f_n(x)|^{1/n} < 1
  • Comparison Test: Compare with known convergent series
  • Alternating Series Test: For (-1)^n f_n(x) where f_n(x) ‚Üì 0

For Uniform Convergence:

Condition: Find M_n such that |f_n(x)| ‚â§ M_n for all x‚ààD, and ‚àëM_n converges

\[|f_{n+1}(x) + f_{n+2}(x) + \cdots + f_m(x)| < \varepsilon\]

Conditions:

  1. Partial sums A_n(x) = ‚àë_{k=1}^n a_k(x) are uniformly bounded
  2. b_n(x) ‚Üì 0 uniformly (monotone decreasing to 0 uniformly)

Test 5: Abel Test for Uniform Convergence

  2. b_n(x) is monotone in n for each x and uniformly bounded

Procedure:

  1. Compute M_n = sup_{x‚ààD}|f_n(x)|
  2. Check if ‚àëM_n converges

Conclusion: If ‚àëM_n converges, we have normal convergence

Trick Category 1: Problem-Solving Strategies

Trick 1: The "Divide and Conquer" Strategy

When a series doesn't converge uniformly on the whole domain, break the domain into parts:

  1. Find where problems occur (usually at boundaries or singular points)
  2. Restrict to subdomains away from problem points
  3. Prove uniform convergence on each subdomain

Example: ‚àëx^n/n converges uniformly on [0,r] for any r<1, but not on [0,1)

Trick 2: The "Œµ/2" or "Œµ/3" Method

When proving uniform convergence from pointwise convergence:

\[|S_n(x) - S(x)| \leq |S_n(x) - S_m(x)| + |S_m(x) - S(x)|\]

Make each term < Œµ/2 using:

  • First term: Cauchy criterion (choose m,n large, independent of x)
  • Second term: Pointwise convergence (m fixed, then n large)

Trick 3: Summation by Parts (Abel Transformation)

For proving Dirichlet/Abel tests:

\[\sum_{k=m+1}^n a_k b_k = A_n b_n - A_m b_{m+1} - \sum_{k=m+1}^{n-1} A_k (b_{k+1} - b_k)\]

where \(A_k = \sum_{j=1}^k a_j\)
